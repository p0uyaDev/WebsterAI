# 🕷️ Mama Webster

<img src="https://github.com/p0uyaDev/WebsterAI/blob/main/assets/MamaWebster.png" width="500" height="500" alt="Mama Webster Logo">

Mama Webster is a Discord AI companion inspired by *The Crypt* community.

She is designed as a warm but firm broodmother personality — offering guidance, presence, and structured interaction inside Discord servers.

This project is currently in early development.

---

## 🚧 Project Status

Active development — core architecture and AI integration in progress.

Planned features:

* Groq-powered AI responses
* Personality system (Broodmother archetype)
* Role-aware behavior (server role sensitivity)
* Modular AI core
* Safe and minimal permission design

---

## 🛠️ Tech Stack

* **Runtime:** Bun
* **Language:** JavaScript (Node.js ecosystem)
* **Discord Library:** discord.js
* **AI Provider:** GroqCloud API
* **Environment Management:** dotenv
* **Linting:** ESLint (or chosen linter)

---

## 🧠 AI Integration

Mama Webster uses GroqCloud’s LLM infrastructure.

Important clarifications:

* All AI processing is handled by Groq’s infrastructure.
* Prompts and responses are subject to Groq’s Terms of Service.
* This project does not store message history unless explicitly implemented in future versions.
* API keys are stored securely in `.env` and are never committed.
* Future contributors must use their own Groq API keys.

This repository does not provide or distribute any API credentials.

---

## 🔐 Permissions Philosophy

Mama Webster follows the principle of least privilege.

She will only request minimal Discord permissions required for:

* Sending messages
* Reading message history
* Using slash commands
* Basic interaction features

No elevated administrative permissions are intended unless explicitly required for moderation features.

---

## 📦 Installation (Development)

```bash
bun install
```

Create a `.env` file:

```
DISCORD_TOKEN=your_discord_token
GROQ_API_KEY=your_groq_api_key
```

Then run:

```bash
bun run start
```
